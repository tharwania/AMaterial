### Installation

It requires [Node.js](https://nodejs.org/)  to run.

Install the dependencies and devDependencies and start the server.

```sh
$ git clone https://github.com/tharwania/AMaterial.git
$ npm install
$ ng serve
```

For production environments...

```sh
$ git clone https://github.com/tharwania/AMaterial.git
$ npm install
$ ng build --prod
```
